About This Method

Hi Daniel, welcome! This course uses the Interchange method, which is designed to get you communicating effectively in English as quickly as possible. Here is some vocabulary that will be useful for your work in Calgary. Many of these are common on a construction site.

  • "Two-by-four" (2x4): A standard piece of lumber. "Can you pass me that two-by-four?"
  • "Level": A tool to see if a surface is perfectly horizontal or vertical. "Make sure the shelf is level before you screw it in."
  • "Blueprint": The detailed plan for a construction project. "Let's check the blueprint again."
  • "Drywall": A type of board used to make interior walls. "We need to install the drywall tomorrow."
  • "To frame a house": To build the basic wooden structure of a house. "We're going to start framing the house next week."
  • "Smoko": (Slang, common in Canada/Australia) A short break for a smoke or a cup of coffee. "Alright crew, time for a quick smoko."
